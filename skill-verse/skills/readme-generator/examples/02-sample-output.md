# the-little-helper

A Rasa Pro 3.14 chatbot that handles customer-service journeys (order tracking,
cancellations, Clubcard replacement, store queries) across WhatsApp, Facebook
Messenger, and webchat.

## Tech stack

- Python 3.10
- Rasa Pro 3.14 / rasa-sdk 3.14
- httpx 0.28.1 (async HTTP)
- Redis ≥ 7.1.0 (token cache)
- presidio-analyzer 2.2.33 (PII masking)
- spaCy en_core_web_sm

## Prerequisites

- Python 3.10
- Docker (for running Redis locally)
- Access to Tesco Nexus PyPI registry (internal)

## Installation

```bash
pip install -r actions/requirements.txt \
  --extra-index-url https://<NEXUS_USER>:<NEXUS_TOKEN>@nexus.ourtesco.com/repository/39_chatbot/simple
```

